Downloader support four types of Youtube url
1.

https://www.youtube.com/watch?v=k3dJKtQmyd0

2.

https://www.youtube.com/watch?v=cqszz_OfAFQ&list=PL0F601FD9BD8B2C43

Fetch links for only single Youtube video identified by the Video Id

3.

https://www.youtube.com/playlist?list=PL0F601FD9BD8B2C43

4.

https://www.youtube.com/watch?v=cqszz_OfAFQ&list=PL0F601FD9BD8B2C43&all=true

This fetch all links for all the videos in Youtube Playlist indentified by Playlist Id
